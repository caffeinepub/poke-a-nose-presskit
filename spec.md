# Specification

## Summary
**Goal:** Overhaul the Press Kit page layout by fixing the game logo size, applying the correct background image, and reordering all content sections into a specified sequence.

**Planned changes:**
- Display the game logo at full content container width while preserving its aspect ratio, fully responsive across all screen sizes
- Apply the same custom background image (and its positioning/sizing/overlay behavior) from the Landing page to the Press Kit page
- Reorder Press Kit page sections to: (1) Game Logo, (2) YouTube Video, (3) About, (4) Game Details, (5) Features, (6) Screenshots, (7) Download All Screenshots button, (8) Socials, (9) Press Email

**User-visible outcome:** The Press Kit page now shows the correct background, a full-width logo, and all content sections displayed in the defined order.
